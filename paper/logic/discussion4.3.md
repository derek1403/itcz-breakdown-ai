# §4.3 的邏輯解說 — Day-1 殘響（manifold shock）到底該怎麼歸因

```說明
這份文件回答你的提問：「條紋真的跟強迫是條狀有關嗎？有些 day-1 特徵根本不是條狀的，
怎麼解釋？」結論先講：你的質疑是對的 —— 正文原本那句「zonally banded because the
forcing itself is zonally elongated」歸因太快、站不住。更站得住的講法是：
**day-1 殘響的幾何不是強迫給的，是「算子在 u0 這一點的線性化 J」給的**；
條狀與非條狀特徵都能從 J 的性質推出來。正文 §4.3 已同步改寫。
```

## 1. 先盤點現象：day-1 到底有哪些東西

我們的 run 在第一次套用 M 之後（n=1），擾動場同時出現至少三類結構：

- **A. 全域緯向條紋**：不只在強迫區，遠離加熱、甚至另一個半球都有，緯向延展、
  近乎與緯圈平行；
- **B. 非條狀的塊狀/偶極異常**：錨定在陸地、地形、海岸線上（之後幾天陸地渦度
  過早成長疑似同源）；
- **C. 棋盤格/高頻格點噪音**：尺度固定在幾個像素，跟任何物理尺度無關。

「強迫是緯向拉長的，所以殘響是條狀的」這句話：解釋不了 A 的**全域性**
（強迫沒碰到的地方為什麼也有條紋？）、完全解釋不了 B 和 C（它們不是條狀）。
所以要換一個統一的歸因。

## 2. 一個先要釐清的數學事實：背景自己的 shock 進不了 u'

有一個直覺講法是「u0 是氣候平均、太平滑、本身就 off-manifold，所以第一步把它
推回流形時甩出條紋」。**這個講法在我們的架構裡不成立**，因為：

$$ u'_n = M(u_0 + \delta_n) - \underbrace{M(u_0)}_{B}. $$

若 $f=0$ 且 $u'_0=0$，則 $u'_n \equiv 0$ **恆等地成立** —— 背景不管多麼
off-manifold，它自己的「被推回流形」全部被 $B$ 精確扣掉了。
所以 day-1 的一切殘響都活在**差分反應** $M(u_0+f)-M(u_0)$ 裡：
是「加了強迫之後，網路反應的改變量」。這點在正文 2.2 的性質 2 已經講了，
在 4.3 的論證裡要再用一次：**條紋不是背景漂移殘留**（不需要跑 f=0 對照，
數學上就是零；但文章裡值得點明，防 reviewer 誤會）。

## 3. 那幾何是誰給的？答案：J 在 u0 這一點的結構

把差分反應展開（正文 2.2 的式子）：

$$ u'_1 = \mathbf{J}\,f + \tfrac12 f^\top \mathbf{H} f + \cdots,
\qquad \mathbf{J} = \left.\frac{\partial M}{\partial x}\right|_{u_0}. $$

day-1 殘響 = 強迫被 $\mathbf{J}$（和曲率項）**投影**出來的東西。
所以殘響的幾何由 $\mathbf{J}$ 的模態結構決定，強迫只決定投影的**係數**。
接下來三類現象就分別對應 $\mathbf{J}$ 的三個性質：

**(A) 全域緯向條紋 ← u0 的近緯向對稱性。**
氣候平均背景幾乎是緯向對稱的。一個（近似）與「沿緯圈平移」可交換的線性算子，
它的本徵/奇異向量（近似）就是**緯向諧波** —— 沿整條緯圈延展的條狀模態。
Pangu 的 attention 是全域的，一步之內就能把局地強迫投影到這些全球模態上。
所以：條紋是**緯向對稱背景的算子模態**，不是強迫形狀的印子。
這一下子解釋了全域性（模態本來就是全球的）、方向（對稱性給的）、
以及「為什麼另一個半球也有」。
可檢驗的預測：把強迫換成圓形斑、或搬到南半球 —— 條紋的方向與全域性
應該**不變**（只有振幅分布變）。若條紋跟著強迫形狀走，這個歸因就錯。

**(B) 非條狀的陸地/地形錨定異常 ← 對稱性在哪裡破，殘響就在哪裡不條狀。**
u0 的緯向對稱被陸地、地形、海岸線這些**尖銳靜態梯度**局地打破，
$\mathbf{J}$ 的敏感度在這些梯度上高度集中（網路學到的就是「這些地方反應大」）。
所以同一個 off-manifold 差分反應，在海上長成條紋、在陸地/地形上長成
錨定的塊狀或偶極 —— 這與 LLAT 專案裡「靜態場偏移在海岸線兩側長出駐留偶極」
是同族機制。B 類特徵不是條紋的反例，是同一機制在對稱破缺處的表現。

**(C) 棋盤格 ← 架構的離散格。**
patch embedding / window attention 有固定像素週期，殘響帶著這個週期
出現，跟物理與強迫都無關 —— 純架構假影，自迴歸會放大它（§3.6 機制二）。

## 4. 「off-manifold」的精確含義（比正文原版更強的一句話）

值得注意：$\mathbf{J}$ 是在 $u_0$ 這一點取的導數，而 $u_0$（平滑氣候平均）
本身就不在訓練分布裡。所以 **off-manifold 的不只是 Taylor 餘項 ——
連線性項 $\mathbf{J}$ 都是在流形外取的導數**，它的模態結構沒有被任何
訓練訊號約束過。這比「餘項被網路曲率主導」更普遍：day-1 殘響即使在
線性階也可以是非物理的（「linear ghost modes」）。
這也給出一個乾淨的判別實驗：對強迫做**振幅掃描**，看 day-1 條紋振幅
隨 $\|f\|$ 的標度 —— 線性標度 ⇒ 殘響活在 $\mathbf{J}$ 裡；
超線性 ⇒ 曲率（$\mathbf{H}$）主導。兩者都是 shock，但機制層次不同。

## 5. 與「物理重力波調整」的區辨（保留原本的檢驗，加強版）

| 檢驗 | 物理慣性重力波調整 | manifold shock |
|---|---|---|
| 時間 | 有頻散時間尺度，數步內演變 | 單步全形出現 |
| 空間 | 從強迫區以群速向外傳 | 錨定不動（條紋在模態上、塊狀在地形上） |
| 幾何 | 由層結與 f 決定的波型 | 由 u0 對稱性 + 靜態梯度 + 架構格決定 |
| 強迫形狀改變 | 波源改變 → 波場改變 | 條紋方向/全域性不變（預測） |
| 振幅標度 | 線性（小振幅時） | 線性（J ghost）或超線性（曲率）都可能 |

另外注意：AI-Forum 早期 run 的 day-1 是「2D 紅藍斑塊」而不是條紋 ——
在這個框架下不矛盾：不同的 IC 處理與強迫（以及可能不同的內插/座標處理）
改變了 $u_0$ 與 $f$，投影出的模態組合不同；斑塊更像 B/C 類。
這反而支持「幾何由算子與背景決定、不由強迫形狀決定」。

## 7. 正文 §4.3 的改寫要點（已執行）

- 刪掉「zonally banded because the forcing itself is zonally elongated」。
- 改為：殘響幾何由算子在 $u_0$ 的線性化決定 —— 近緯向對稱背景的全球
  準緯向模態（條紋，包括遠離強迫處）＋ 尖銳靜態特徵上的錨定異常
  （非條狀、陸地/地形上）＋ 架構格週期（棋盤格）；強迫決定投影係數，
  不決定幾何。
- 加一句「off-manifold 的不只是餘項，J 本身也是」。
- 現象敘述改為「條紋為主、伴隨陸地錨定的非條狀成分」，不再宣稱全部是條狀。
