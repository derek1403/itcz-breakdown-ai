## 2. 成功的加熱設計（詳細）

加熱 `f` 只加在**溫度**上，3D 形狀 = 水平 `H(lat,lon)` × 垂直 `V(p)` × 振幅。

### 2a. 水平 `H(lat,lon)` — 論文式 `perturbations_ellipse`
`perturbations_ellipse(lat, lon, k=8, ylat=10, xlon=195, L=25000 km)`，峰值 1.0。
- **緯向（南北）**：`cos(k·(φ−ylat))` 第一個波瓣，峰值在 `ylat=10°N`，到 `φ = ylat ± 90°/k` 歸零。
  `k=8` → 約 **−1.25°~21.25°N**（k 越大越窄）。
- **經向（東西）**：Gaspari-Cohn (eq.3)，中心 `xlon=195°E (=165°W)`，離中心 `d=L` 公里處歸零；
  `L=25000 km` → 範圍約 **60°E ~ 30°W（330°E）**，涵蓋整條太平洋 ITCZ。
- 調整旋鈕：`XLON`=中心經度、`L_KM`=寬度（半寬° × 111.3 km）、`K_WAVE`=南北寬度、`YLAT`=南北中心。

### 2b. 垂直 `V(p)` — Deep（deep convective）
`V(p) = sin(π·(p−200)/800)`，只在 **1000–200 hPa** 非零、**峰值 600 hPa**、1000/200 hPa 歸零。

### 2c. 振幅
**2.5 K/day**（24h 模型 → 等於 2.5 K/step）。

### 2d. 初始場（IC）= JAS
**1979–2019 JAS（7–9月）氣候平均**，`ref/ERA5_means/mean_JAS.h5`，由 `config.yml` 的
`mean_state_season: JAS` 控制。夏季北半球 ITCZ 正好在 10°N，是 Deep 加熱能起作用的關鍵。
加熱與 IC 確認圖：[figs/heating_dist.png](figs/heating_dist.png) · [figs/ic_JAS.png](figs/ic_JAS.png)

### 2e. 跑法（semi-linear / make_steady）
每步：跑 24h Pangu → `out_pl[2] += heat3d`（加熱進溫度）→ `out_pl −= mean_pl_dt`、
`out_sfc −= mean_sfc_dt`（扣穩態背景趨勢，使背景不動）→ 下一步。存的是**擾動** = 場 − JAS 平均。

## 3. 怎麼用這些 code

```bash
cd /wk2/pc/AI_models/itcz-breakdown-ai/investigation/aiforum_repro
PY=/home/pc/.conda/envs/pangu_env/bin/python

# (0) 先畫加熱分布 + JAS IC 確認圖（改 0_plot_heating.py 頂部參數後重跑即可預覽）
$PY 0_plot_heating.py            # -> figs/heating_dist.png, figs/ic_JAS.png

# (1) 跑主結果：JAS + Deep + 2.5 K/day
$PY 1_run_pangu.py Deep          # -> output/aiforum_JAS_Deep_amp2.5.h5

# (2) 畫渦度/TCWV/時序 panels（自動讀所有 output/aiforum_*_amp*.h5）
$PY 2_plot_results.py            # -> figs/vort_*.png, tcwv_*.png, timeseries.png
```

### 變體
```bash
# inverse：一開始把 IC+tendency 的 lat 方向 [::-1] 翻轉再丟 Pangu
$PY 1_run_pangu.py Deep inverse        # -> ..._inverse.h5

# DJF（複製檔，season 固定 DJF，其餘同）
$PY 1_run_pangu_djf.py Deep
$PY 1_run_pangu_djf.py Deep inverse

# 垂直 uniform（含低層）也可：把第一參數 Deep 換成 uniform
$PY 1_run_pangu.py uniform
```

## 6. 已知限制
- ζ 峰 ~40 略低於目標 47（同量級；若要更貼可把 amp 微調到 ~2.8–3.0）。
- ITCZ breakdown 是**混沌**，逐日確切渦旋位置不可逐點重現，只能對齊統計/定性特徵。
- day1 仍有全域微弱緯向細結構（脈衝加熱激發的重力波）；中緯度後期較活躍。
